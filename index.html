Artivist Protocol DAO

ARTIVIST é uma plataforma web3 de crowdfunding que transforma a colaboração entre artistas, ativistas e organizações sociais, criando uma cultura e economia para empoderar colaboradores. Focando em public goods, nosso objetivo é usar a tecnologia web3 para revolucionar o financiamento, promovendo ganhos mútuos para todos os stakeholders.
